# Catch_Mind
Android Project
개발날짜 2017/11~2017/12
------------------------------------------------------

1. 프로젝트 제작배경
- 통신량이 많은 TCP/IP 통신과 AWS를 이용한 웹 서버 구축 및 http통신을 이용한 DB제어 목적.

2. 프로젝트 벤치마킹
- 넷마블 CatchMind

3. 기본 게임 방법
- 회원가입을 통한 아이디 생성
- 생성된 아이디로 게임에 접속 시 바로 게임방에 입장.
- 맨 처음 게임방에 접속한 유저가 출제자가 된다.
- 출제자는 그림을 그리려서 다른 플레이어들이 맞출 수 있게 한다.
- 정답을 맞춘 플레이어는 출제자가 된다.
